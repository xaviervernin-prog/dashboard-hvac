# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| main    | ✅        |

## Reporting a Vulnerability

**Ne pas ouvrir d'issue publique pour une faille de sécurité.**

Envoyer un rapport privé via GitHub Security Advisories :  
**Security → Report a vulnerability** dans ce repository.

Inclure :
- Description de la vulnérabilité
- Étapes pour reproduire
- Impact potentiel
- Suggestions de correction si possible

Une réponse sera apportée sous 72h.
